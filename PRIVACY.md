# Privacy Policy

local-md-editor is designed with a local-first approach.

## Data Collection

This extension does NOT:

- collect personal data
- send telemetry
- track user activity
- upload files or content
- communicate with external servers

All editing happens locally inside VS Code.

## Network Access

This extension does not perform external network requests.

No analytics, cloud sync, or third-party APIs are used.

## File Access

The extension only accesses files that the user explicitly opens inside VS Code.

No background scanning or indexing is performed outside the active workspace.

## Third-Party Dependencies

Dependencies are kept minimal and reviewed before inclusion.

## Security Philosophy

The goal of this project is to provide:

- local-first editing
- privacy-focused design
- transparent behavior
- minimal permissions

## Contact

If you discover a security issue or privacy concern, please open an issue on GitHub.
